PRD / Technical Roadmap

Lumia Graph-RAG Core

将书籍知识图谱与信号图谱搜索统一到同一套 Graph-RAG 思路中：文本像书籍目录一样组织，信号像多尺度章节一样编码，再通过图谱召回、向量检索和局部精排实现精准、可解释、低算力的搜索。

项目目标

书籍知识图谱

面向一万本 Markdown 电子书，建立书、卷、章、段落、概念、引用关系，用于大规模精准 RAG。

信号图谱搜索

面向信号、频谱图和特征图，建立时间窗口、局部模式、频谱结构和相似关系，用于低算力相似匹配。

Gradio Lab

作为研发调参实验台，比较 BM25、向量、图谱过滤、局部卷积、DTW 与 rerank 效果。

检索流程

书籍检索

Query → 实体链接 → Wiki 召回 → 书籍召回 → 章节召回 → chunk 混合召回 → rerank → 多书覆盖 → 引用回答。

信号检索

Query Signal → 预处理 → 分段 → 特征提取 → ANN 召回 → 图谱过滤 → 局部卷积 / DTW 精排。

融合评分

向量相似度、频谱相似度、局部匹配、图谱上下文和元数据共同决定最终排序。

MVP 里程碑

阶段范围验收指标
MVP-1 书籍图谱100 本 Markdown，章节切分，FTS + 向量召回综述模式返回 15 本以上不同来源
MVP-2 信号图谱1000 条信号，多尺度 segment，ANN + 局部精排相比全库卷积耗时下降 70% 以上
